About me

Hello! I'm a self-taught frontend web developer based in Barcelona, Spain. I primarily work with CSS and HTML with Vanilla Javascript to provide user interactions. I'm also familiar with UX/UI design and consider it to be my hobby.

My main priority is accessibility. I also pay a lot of attention to responsive images and other ways to make my websites works on any devices. My technical approach includes creating levels of abstraction with SASS and using modern CSS techniques.

My work is best described with the projects in my portfolio.
